Program to obtain the information about the [A]Host [B]Network


[C]Protocols [D]Domains 
CODING:
import java.lang.*;
import java.io.*;
import java.net.*;
class udl{
public static void main(String args[])throws MalformedURLException{
URL url=new URL("http://www.yahoo.com");
try{
System.out.println("Host name is"+url.getHost());
System.out.println("Port no is"+url.getPort());
System.out.println("Protocol used is"+url.getProtocol());
}
catch(Exception e)
{
System.out.println("Error="+e);
}
}
}



OUTPUT:
Host name is www.yahoo.com
Port no is -1
Protocol used is http

Ex:No:2 Demonstrate program to manipulate the IP Address

CODING:
import java.util.*;
import java.lang.*;
import java.net.*;
public class getOwnIP{
public static void main(String args[])throws UnknownHostException{
try{
InetAddress IPO=InetAddress.getLocalHost();
System.out.println("IP of the system="+IPO.getHostAddress());
System.out.println("Canonical Hostname="+IPO.getCanonicalHostName());
System.out.println("Hostname ="+IPO.getHostName());
System.out.println("AllByName="+IPO.getAllByName(IPO.getHostName()));
System.out.println("Loopback Address="+IPO.getLoopbackAddress());
System.out.println("Address="+IPO.getAddress());
System.out.println("isAnyLocal Address="+IPO.isAnyLocalAddress());
System.out.println("isMulticast Address="+IPO.isMulticastAddress());
}
catch(Exception e)
{
System.out.println("Exception caught="+e.getMessage());
}
}
}
